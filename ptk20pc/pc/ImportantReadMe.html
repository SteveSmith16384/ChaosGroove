Microsoft Windows

You also need to add these libraries to your project:
winmm.lib ( joystick support )
opengl32.lib ( opengl )
shell32.lib ( URL / File Launch )
wininet.lib ( http requests )
dsound.lib ( sound with KSound)
dxguid.lib ( sound with KSound )
ddraw.lib ( directdraw )
d3dx.lib ( note that you must use d3dx.lib that comes with the dx7 sdk, if you use the one that comes with DirectX 9 it will ask for a D9.dll ...)


openGL official website openGL SDK : http://www.opengl.org


For Microsoft Visual C++ libs are located in the
C:\Program Files\Microsoft Visual Studio\VC98\Lib\ folder

Note that the new ptk is compiled in multithreaded mode. ( the single threaded would cause problems on hyper threaded computers ) we recommend that you follow the step by step instructions on how to set up the project:
http://www.laughingd.com/ptk_dotnetsetup.aspx

( paths may vary according to your default installation of the software )
